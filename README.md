# Be My Valentine

A playful React app that asks, "Will you be my Valentine?" The "No" button dodges your cursor or touch to keep things lighthearted.

## Features

- Centered Valentine prompt with a cheerful visual theme.
- "Yes" button shows a positive response.
- "No" button moves on hover, touch, and click while staying inside its bounds.
- Mobile-friendly layout with no scrolling needed.

## Tech Stack

- React (Create React App)
- Vanilla CSS

## Getting Started

Install dependencies:

```
npm install
```

Start the dev server:

```
npm start
```

The app runs at http://localhost:3000.

## Scripts

- `npm start` - Run the app locally.
- `npm test` - Run tests in watch mode.
- `npm run build` - Build a production bundle.
- `npm run deploy` - Publish to GitHub Pages.

## Deploy to GitHub Pages

1. Ensure the repository is on GitHub.
2. Confirm `homepage` in `package.json` matches your GitHub Pages URL.
3. Run `npm run deploy`.

## Notes

This app is client-side only and does not use any backend services.
